Game

The game module contains various functionality related to the game.

Note

Make sure to call crazy_init before stating to use the extension.

Functions

The following module functions are available:

  • crazy_game_settings
  • crazy_game_gameplay_start
  • crazy_game_gameplay_stop
  • crazy_game_loading_start
  • crazy_game_loading_stop
  • crazy_game_happytime
  • crazy_game_is_instant_multiplayer
  • crazy_game_invite_link
  • crazy_game_get_invite_param
  • crazy_game_show_invite_button
  • crazy_game_hide_invite_button

crazy_game_settings

This function retrieves the game settings object from CrazyGames SDK.


Syntax:

crazy_game_settings()




Returns:

Struct


Example:

var settings = crazy_game_settings();
show_debug_message("Game settings: " + json_stringify(settings));

crazy_game_happytime

This function triggers CrazyGames' "happy time" event, indicating a positive gameplay moment. Can be called on various player achievements (beating a boss, reaching a highscore, etc.). It makes the website celebrate (for example by launching some confetti). There is no need to call this when a level is completed, or an item is obtained.


Syntax:

crazy_game_happytime()




Returns:

N/A


Example:

crazy_game_happytime();

crazy_game_invite_link

This function creates a multiplayer invite link based on the provided parameters.


Syntax:

crazy_game_invite_link(_params)
Argument Type Description
_params String A JSON-formatted string containing invite link parameters.




Returns:

String or Undefined

The generated invite link, or undefined if not initialised.


Example:

var inviteParams = { roomId: 12345 };
var link = crazy_game_invite_link(json_stringify(inviteParams));
show_debug_message("Invite link: " + link);

crazy_game_get_invite_param

This function retrieves a specific invite parameter from the CrazyGames SDK.


Syntax:

crazy_game_get_invite_param(key)
Argument Type Description
key String The parameter key to retrieve.




Returns:

String or Undefined

The parameter value, or undefined if not initialised.


Example:

var roomCode = crazy_game_get_invite_param("roomId");
if (roomCode != undefined) {
    show_debug_message("Joined room: " + string(roomCode));
}

crazy_game_show_invite_button

This feature allows you to display a button in the game footer, that opens a popup containing the invite link.

Warning

The invite button should only be used to invite players to a multiplayer gaming session. Please avoid using it for other use cases, such as a Share button for example, as this may lead to a delayed submission check or even game rejection.


Syntax:

crazy_game_show_invite_button(params)
Argument Type Description
params String A JSON-formatted string defining link parameters.




Returns:

String


Example:

// the returned link looks the same as the link
// returned by the crazy_game_invite_link method
var link_params = {
    roomId: 12345,
    param2: "value",
    param3: "value",
};
var link = crazy_game_show_invite_button(json_stringify(link_params));
