Banner

The banner module contains functionality for displaying banners within your game.

Note

Make sure to call crazy_init before stating to use the extension.

Functions

The following module functions are available:

  • crazy_banner_request_banner
  • crazy_banner_request_responsive_banner
  • crazy_banner_clear_banner
  • crazy_banner_clear_responsive_banner
  • crazy_banner_clear_all_banners

Constants

The following constants are available:

  • CRAZY_BANNER_POSITION

crazy_banner_request_banner

This function requests a static banner ad from the CrazyGames SDK.


Syntax:

crazy_banner_request_banner(banner_id, width, height, position, callback_success, callback_failed)
Argument Type Description
banner_id Real Unique ID for the banner (0-9).
width Real Banner width in pixels.
height Real Banner height in pixels.
position CRAZY_BANNER_POSITION Banner position.
callback_success Function Called when the banner loads successfully.
callback_failed Function Called if the banner fails to load, with error info.




Returns:

N/A


Example:

crazy_banner_request_banner(
    0,
    300, 250,
    CRAZY_BANNER_POSITION.CENTER_MIDDLE,
    function() { show_debug_message("Static banner loaded!"); },
    function(err) { show_debug_message("Banner failed: " + json_stringify(err)); }
);

crazy_banner_request_responsive_banner

This function requests a responsive banner ad from the CrazyGames SDK.


Syntax:

crazy_banner_request_responsive_banner(banner_id, width, height, position, callback_success, callback_failed)
Argument Type Description
banner_id Real Unique ID for the banner (0-9).
width Real Reserved width for banner positioning.
height Real Reserved height for banner positioning.
position CRAZY_BANNER_POSITION Banner position.
callback_success Function Called when the banner loads successfully.
callback_failed Function Called if the banner fails to load, with error info.




Returns:

N/A


Example:

crazy_banner_request_responsive_banner(
    0,
    160, 600,
    CRAZY_BANNER_POSITION.CENTER_MIDDLE,
    function() { show_debug_message("Responsive banner loaded!"); },
    function(err) { show_debug_message("Responsive banner failed: " + json_stringify(err)); }
);

crazy_banner_clear_banner

This function clears a static banner ad from the screen and from the CrazyGames SDK.


Syntax:

crazy_banner_clear_banner(banner_id)
Argument Type Description
banner_id Real Unique ID of the banner to clear (0-9).




Returns:

N/A


Example:

CRAZY_BANNER_POSITION

This enumeration holds the different banner positioning constants for CrazyGames banners.

These constants are referenced by the following functions:

  • crazy_banner_request_banner
  • crazy_banner_request_responsive_banner


Member Description
LEFT_TOP Top-left corner of the game area.
CENTER_TOP Top-center of the game area.
RIGHT_TOP Top-right corner of the game area.
LEFT_MIDDLE Center-left side of the game area.
CENTER_MIDDLE Exact center of the game area.
RIGHT_MIDDLE Center-right side of the game area.
LEFT_BOTTOM Bottom-left corner of the game area.
CENTER_BOTTOM Bottom-center of the game area.
RIGHT_BOTTOM Bottom-right corner of the game area.
